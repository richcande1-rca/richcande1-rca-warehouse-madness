# Warehouse Madness

Phone-first arcade prototype for a simple dock-door loading game.

## Prototype loop

- Pallets appear on the staging floor.
- Tap a numbered pallet.
- Tap the matching dock door.
- Forklift loads the pallet.
- Trailer pallet count increases.
- Win by filling all four trailers.
- Lose if the staging floor fills up.

## Files

- `index.html.txt` — prototype file. Rename to `index.html` for GitHub Pages.
- `README.md` — this file.

## First design constraints

- Phone-first touch targets.
- No dependencies.
- No canvas preview needed.
- Simple grey floor, yellow railings, four doors, one forklift.
